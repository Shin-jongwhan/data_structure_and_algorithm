## 표현 방법
- O(1) : 시간복잡도가 항상 같은 것
- O(N) : 시간복잡도가 단계가 상승할 때마다 증가하는 것
- O(logN) == O(log₂N) : 컴퓨터 과학에서는 밑이 2인 log 와 그냥 log 를 같은 것이라 한다. 이진 검색이 O(logN) 에 해당한다.
